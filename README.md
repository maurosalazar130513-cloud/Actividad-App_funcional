Actividad Aplicacion web funcional

Este proyecto es una aplicación web para gestión de usuarios y autenticación, desarrollada con Node.js, Express, MongoDB y un frontend en React.

Requisitos previos
- Node.js
- npm
- MongoDB

Instalación del proyecto

Clonar el repositorio:
https://github.com/maurosalazar130513-cloud/Actividad-App_funcional

Instala las dependencias en app_node y app_react:
npm install

Para iniciar el servidor:

- En app_node
npm run dev

- En app_react
npm start
